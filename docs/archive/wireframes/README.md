# Legacy Wireframes Archive

This directory contains historical design wireframes and mockups that have been superseded by the current **Medical AI Audit Ledger** and **Behavioral Verification** platform.

These artifacts are preserved for historical reference and to provide context on the early "wallet-centric" project direction. They do **not** represent the current UI or product strategy.

## Archived Files
- **register-agent-2024-04.pdf**: Early wireframes for agent identity registration.
- **verify-agent-2024-04.pdf**: Early wireframes for blockchain-backed fingerprint verification.

For the current design direction, please refer to the live components in:

- `src/components/MedicalAuditDashboard.tsx`
